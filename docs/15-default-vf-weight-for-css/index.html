This is a test of default font weights for Encode Sans (VF).

The VF currently has a default value of 100. However, the browser still displays 400 weight type for a <p> tag, unless told otherwise.

The CSS looks like this:

        @font-face {
            font-family: "Encode Sans";
            font-weight: 100 900;
            src: url("./EncodeSans-Thin.woff2") format("woff2");
        }
        
        * {
            font-family: "Encode Sans";
            font-size: 24px;
            max-width: 50ch;
        }
        
        .weight100 {
            font-weight: 100;
        }
